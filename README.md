# PencatatanKalori
Baca dokumentasi dan cara penggunaan pada file [Dokumentasi-Aplikasi-Pencatatan-Kalori.pdf](https://github.com/xSteins/PencatatanKalori/blob/update-dokumentasi/Dokumentasi-Aplikasi-Pencatatan-Kalori.pdf). 

Aplikasi dibangun menggunakan daftar pustaka berikut: 
* Jetpack Compose UI (Tampilan antarmuka)
* Jetpack Navigation (`navigationCompose` untuk navigasi layar)
* Room Database (Basis Data)
* Hilt (viewmodel injection)
* Coil (manajemen gambar)

## Kebutuhan Development

- **Android Studio** (Dikembangkan menggunakan versi meerkat dan ladybug)
- **Windows** 11
- **Perangkat pengujian (Debug Device)**: Minimal API 29 (Android 10)

## Cara Compile

1. Buka project di Android Studio
2. Jalankan **Gradle Sync** (File > Sync Project with Gradle Files)
3. Buat konfigurasi debug: Run > Edit Configurations > + > Android App
4. Jalankan aplikasi di device atau emulator

## Cara Menggunakan Aplikasi
1. Buka tab [Releases](https://github.com/xSteins/PencatatanKalori/releases) di GitHub
2. Download file APK terbaru
3. Install APK di device Android Anda
4. Buka aplikasi dan isi data pribadi (berat badan, tinggi, umur, jenis kelamin)
5. Pilih tingkat aktivitas dan tujuan: menaikkan massa otot (bulking) atau menurunkan berat badan (cutting).
