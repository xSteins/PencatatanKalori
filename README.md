# PencatatanKalori

Aplikasi Android untuk mencatat asupan kalori harian dan kalori yang terbakar.

## Kebutuhan Development

- **Android Studio** (versi terbaru)
- **Windows** (atau Linux/macOS)
- **Debug Device**: Minimal API 29 (Android 10)

## Cara Compile

1. Buka project di Android Studio
2. Jalankan **Gradle Sync** (File > Sync Project with Gradle Files)
3. Buat konfigurasi debug: Run > Edit Configurations > + > Android App
4. Jalankan aplikasi di device atau emulator

## Cara Install

1. Buka tab [Releases](https://github.com/xSteins/PencatatanKalori/releases) di GitHub
2. Download file APK terbaru
3. Install APK di device Android Anda

## Cara Menggunakan Aplikasi

1. Buka aplikasi dan isi data pribadi (berat badan, tinggi, umur, jenis kelamin)
2. Pilih tingkat aktivitas dan tujuan (turun/naik/jaga berat badan)
3. Catat makanan dengan tombol **Add Food**
4. Catat aktivitas olahraga dengan tombol **Add Workout**
5. Lihat riwayat di tab **History**
6. Ubah data profil lewat tab **Profile**
